# Orion-Context-Engine
Ecossistema de microsserviços voltado à gestão inteligente de fomento acadêmico. Integra um agente de IA contextual (Python/Gemini) capaz de interpretar linguagem natural, consultar contexto institucional e operar, com segurança, uma API de missão crítica (Java/Spring Boot), com interface reativa em Angular.
